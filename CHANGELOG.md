# Changelog

All notable changes to ResumeForge are documented here.
This project follows [Semantic Versioning](https://semver.org) (MAJOR.MINOR.PATCH).

## [1.0.0] - 2026-07-01

First production release.

### Added
- Three-panel builder: live content editor, exact A4/Letter preview, ATS analysis.
- Four ATS-safe single-column templates: Software Engineering, UI/UX Design, Graphic Design, Researcher/Academic.
- Three starter profiles: Experienced, Fresher, Study Abroad.
- Predefined keyword chips (skills, languages, degrees, job titles) with a Custom option everywhere.
- Add / reorder (drag handle + arrows) / hide / delete sections, including custom section types.
- Live ATS readiness score with a prioritised fix-it checklist and template-aware suggested keywords.
- Tailor to JD: offline keyword engine that extracts job role, location, skills, and other keywords from a pasted job description and flags matches vs. gaps.
- Real-text print-to-PDF (selectable, ATS-parseable) for A4 and Letter.
- Light / dark interface themes; Inter UI font.
- Optional theme-matched LinkedIn / GitHub icons next to links.
- Named save/load (up to 50 resumes) in local storage, plus JSON export/import.
- Memory-foam textarea heights that persist across edits and reloads.
- Manually editable zoom (25%-300%).
- 3D button styling, capsule top-bar controls, RF logo + favicon, centered footer.
